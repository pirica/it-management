# Modules with File Upload Functionality

The following modules in the IT Management application include "Choose File" or "Submit File" (file upload) functionality:

1. **Calendar**
   - **Path:** `modules/calendar/index.php`
   - **Description:** Allows importing events from an ICS file.

2. **Employees**
   - **Path:** `modules/employees/index.php`
   - **Description:** Supports importing employee data from Excel (.xlsx, .xls) or CSV files via a client-side parser.

3. **Equipment**
   - **Path:** `modules/equipment/create.php`
   - **Description:** Allows uploading one or more photos during equipment creation or editing.

4. **Events**
   - **Path:** `modules/events/index.php`
   - **Description:** Provides functionality to import events from an ICS file.

5. **Explorer**
   - **Path:** `modules/explorer/index.php`
   - **Description:** A file manager that allows uploading multiple files via a drag-and-drop area or file input.

6. **Floor Plans**
   - **Paths:** `modules/floor_plans/create_upload_view.php`, `modules/floor_plans/gallery_index_view.php`
   - **Description:** Supports uploading images, PDFs, or AutoCAD files for floor plans.

7. **Patches & Updates**
   - **Paths:** `modules/patches_updates/create.php`, `modules/patches_updates/edit.php`, `modules/patches_updates/index.php`, `modules/patches_updates/list_all.php`, `modules/patches_updates/view.php`
   - **Description:** Includes photo upload functionality for patch records across various views.

8. **Settings**
   - **Path:** `modules/settings/index.php`
   - **Description:** Allows uploading a favicon image and importing database state from a SQL file.

9. **Tickets**
   - **Path:** `modules/tickets/create.php`
   - **Description:** Supports uploading multiple photos when creating a new ticket.
